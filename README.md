# TATA-AIA Advisor

A modern insurance policy advisor website with contact forms and brochure downloads.

## Features

- Responsive design with smooth scrolling navigation
- Policy modals with detailed information
- Contact form with Supabase integration
- Secure PDF brochure downloads with download tracking
- Mobile-friendly interface

## Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- Supabase for backend services
- Node.js/Express backend for secure API routes

## Local Development

1. Clone the repository
2. Open `index.html` in a browser (or use a local server)
3. For full functionality, run the backend server

## Backend Deployment

The backend is deployed separately and handles:
- Secure PDF URL retrieval
- Download count tracking
- Lead form submissions
